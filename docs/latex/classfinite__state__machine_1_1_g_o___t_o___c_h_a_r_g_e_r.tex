\doxysubsection{Detailed Description}
\begin{DoxyVerb}This state is triggered as a result of the message received from GO_IN_ROOM state when battery is low.
In this state *target_position* becomes room *#E* and *update_ontology()* function updated and room #E
add as a first room in **URGENT CLASS**. if room #E is reached state machine changes to WAIT_FOR_CHARGING.\end{DoxyVerb}
